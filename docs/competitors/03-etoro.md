## 2. Sosyal Trading & CopyTrader
- **Mekanik:** yatırımcı seç, fon ayır → hesabın mevcut + gelecek tüm işlemlerini gerçek zamanlı aynalar. Replikasyon orantılı.
- Limitler: min copy **$200**/yatırımcı, min pozisyon $1, max $500,000, aynı anda max **100 yatırımcı**.
- **Copy Stop-Loss:** kayıp eşiğinde durur. Default %5.
- Copy ücreti YOK (sadece underlying maliyet: spread, %1 kripto, overnight).
- **US kısıtı:** copy trading sadece KRİPTO (SEC).
- **Popular Investor (PI) — 4 kademe:**
  - Cadet: ≥2 ay, equity ≥$1,000, ≥1 copier → sadece tanınma.
  - Champion: equity ≥$10,000, ≥5 copier, AUC ≥$50k → **$400/ay** (büyürse $800).
  - Elite: equity ≥$25,000, AUC ≥$500k → **%1.5 AUC/yıl ($10M'a kadar)**.
  - Elite Pro: equity ≥$50,000, AUC ≥$10M → %1.5 ($20M'a kadar) + %0.5 üstü.

## 6. Trading Özellikleri
- Order: Intl — Market, Limit, SL/TP/Trailing. US — Market/Limit/Stop/Stop-limit, TIF Day/GTC.
- Kaldıraç (EU/UK retail, ESMA): forex major 30:1, hisse/ETF 5:1, kripto 2:1. **US: 1:1.**
- ESMA koruması: %50 margin close-out, negatif bakiye koruması.
- SL/TP: kaldıraçlı pozisyonda zorunlu. Trailing Stop-Loss.
- CFD maliyet: overnight (gecelik), **weekend = 3× overnight**.
- **Charts: TradingView Advanced ("ProCharts")** — 100+ indikatör, 50+ çizim, multi-timeframe.
- One-Click Trading. Native desktop YOK (web).
